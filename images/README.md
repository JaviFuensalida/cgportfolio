## Recomendaciones

- **Formato:** JPG o PNG
- **Resolución:** Mínimo 1920px de ancho para mejor calidad
- **Optimización:** Comprimir imágenes para web (70-85% calidad)
- **Aspect Ratio:** Las portadas funcionan mejor con ratio 4:5 (vertical)

Si las imágenes no están disponibles, se mostrará un placeholder automáticamente.
